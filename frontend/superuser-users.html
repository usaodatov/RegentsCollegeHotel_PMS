Superuser – Manage Staff Users

Use this page to create and delete staff accounts. For now, responses will show as raw JSON after submit – just use your browser’s Back button to return to this page.

Create staff user

This will send a POST /api/users with username and password.

Delete staff user

This will send a POST /api/users/delete with user_id. You can grab the ID from the “View all users” endpoint below.

View all users

Opens the raw JSON list of users returned by your API. Use this to confirm that create/delete is working.
